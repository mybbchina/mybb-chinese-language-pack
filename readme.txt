MyBB是一个基于PHP+MySQL搭建，功能强大，高效的开源论坛系统。

语言包的安装：
1.将以上文件夹复制到论坛根目录覆盖同名文件夹和同名文件
  注意:如果您只想添加中文语言包(并不想将中文语言作为论坛主要语言),请只复制inc/language和images文件夹.
2.选择论坛语言默认为中文(简体)
3.后台设置(Configuration)-全局设置(Settings)-语言设为中文(简体)

完整安装包的安装：
1.使用二进制方式将upload目录上传至服务器.
2.在浏览器地址栏输入http://www.yourdomain.com/upload/install 敲回车，强烈建议将upload文件夹改名之后再安装
3.按照向导安装MyBB中文版

升级：
1.备份论坛所有文件和数据（作任何操作之前都强烈推荐备份）
2.上传新版本的文件至你的论坛目录覆盖
3.在浏览器地址栏输入http://www.yourdomain.com/upload/install 敲回车，选择升级
4.选择您需要从什么版本升至当前最新版本
5.按照向导升级MyBB中文版

注：upload可以改为您想要的名称，如果域名根目录就是论坛程序的话，如：http://www.yourdomain.com/ 或 http://bbs.yourdomain.com/
则不要上传整个upload目录，而是上传upload目录中的文件，然后在浏览器地址栏输入：http://www.yourdomain.com/install 或 http://bbs.yourdomain.com/install

2009.4.19
MyBB 1.4.5 发布 - 维护和安全发布
1.后台修复一个低XSS漏洞，标记为低是因为它需要系统管理员权限。这个安全漏洞的发现由ketto93报告 
2.后台修复一些低CSRF漏洞，这些都是低优先级是因为它们只出现在极为罕见的情况下，并且不会危及任何信息。它们只能用于协助DDOS攻击。这些漏洞被内部发现并修复。
3.修复一个生成帖子密钥时算法上的小缺点。低优先级，因为它需要大规模的电脑运算才能成为一个问题。这一缺点被frostschutz发现并报告。
4.从MyBB1.4开始，逗号不允许用在用户名中，MyBB1.4.5将强制清除用户名中的逗号，请在您的论坛告知这一变化。
5.大量的速度和稳定性方面的改进，以负载更大型的论坛。
6.很多其它漏洞的修正。
